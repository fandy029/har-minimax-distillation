# KuHar 数据集详解

## 基本信息

| 项目 | 值 |
|------|----|
| 全称 | Korean Human Activity Recognition |
| 来源 | 韩国梨花女子大学 |
| 传感器 | 腰部佩戴 IMU (加速度3轴 + 陀螺仪3轴 + 2额外通道) |
| 采样率 | 100 Hz |
| 原始通道 | 8 列 (col0:acc时间戳, col1-3:acc x/y/z, col4:gyro时间戳, col5-7:gyro x/y/z) |
| 窗口大小 | 128步 (1.28秒) |
| 步进 | 64步 (50%重叠) |
| 训练/验证/测试 | 80% / 10% / 10% |
| 总训练窗口 | 57,384 |

## 蒸馏建议

- 优先用 Lay(5), Run(14), Jump(8) 验证蒸馏有效性 (准确率>80%)
- Walk系列(11-13)和Stair系列(15-16)可能需要合并类
- correct_only版提供蒸馏效果上界
- 筛选版在数量和质量间折中
