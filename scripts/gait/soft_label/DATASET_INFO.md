# Gait 数据集

## 基本信息

| 项目 | 值 |
|------|----|
| 来源 | 自采数据集 |
| 传感器 | 3轴加速度计 (frontal/vertical/lateral) |
| 采样率 | 40 Hz |
| 滤波 | 10Hz uniform_filter1d 提取重力方向 |
| 窗口大小 | 128步 (3.2秒) |
| 步进 | 64步 (50%重叠) |
| 总训练窗口 | 673 |
| 类别 | 4 |
| 软标签准确率 | 79.8% |

## 4 类详情

| ID | 类名 | 窗口数 | 类型 | 难度 |
|----|------|--------|------|------|
| 0 | sit_on_bed | 161 | 静态坐姿 | 易(与chair相似) |
| 1 | sit_on_chair | 38 | 静态坐姿 | 难(样本少) |
| 2 | lying | 436 | 静态躺姿 | **极简单**(gve~0.1独有) |
| 3 | ambulating | 38 | 动态走路 | 难(样本少) |

## 区分规则

- **lying vs 其他**: gve<0.2 → lying (唯一)  
- **ambulating vs 静态**: free_mag_std>0.10 + peaks>10 → ambulating
- **sit_on_bed vs sit_on_chair**: gfr<0.50 → bed, gfr>0.50 → chair
